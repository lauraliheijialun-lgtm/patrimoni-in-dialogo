Componenti implementati

Questa pagina serve per l’orale: elenca i componenti HTML/CSS/JS usati e il loro ruolo nel progetto.

Navbar responsive con menu mobile, dropdown degli item e link a tutte le pagine.

Card per la navigazione del catalogo; ogni card contiene immagine, status, tema e pulsante verso la scheda item.

Filtro JavaScript per catalogo e tabella dati.

Finestra modale nella home per spiegare l’idea progettuale senza abbandonare la pagina.

Componente espandibile usato qui per descrivere gli elementi tecnici.

Usati nelle schede item per distinguere descrizione, relazione, metadata e fonti; slideshow per la visualizzazione iconografica.

Integrazione di OpenStreetMap tramite iframe; file CSV/JSON/XML usabili come dati del progetto; GitHub Pages previsto per hosting.

Per la presentazione: puoi dire che il sito usa HTML5, CSS, JavaScript vanilla, metadati Dublin Core in head, XML/DC, un esempio TEI e riconciliazione tramite UNESCO/Wikidata/GeoNames.
